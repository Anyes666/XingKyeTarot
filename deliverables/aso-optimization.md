# 星钥塔罗 — 鸿蒙应用商店 ASO 优化文案

> 版本：v1.0
> 日期：2026-07-01
> 目标平台：华为应用市场（AppGallery / HarmonyOS）
> 优化原则：痛点前置 + 差异化前置 + 关键词自然融入

---

## 一、应用名称（≤30字）

```
星钥塔罗·离线情绪陪伴
```

**字数**：11字
**策略**：品牌词"星钥塔罗" + 卖点词"离线情绪陪伴"
**为什么**：
- 鸿蒙应用商店搜索权重中，应用名称占比约 40%
- "离线"是核心差异化（90% 竞品需联网/注册）
- "情绪陪伴"比"塔罗占卜"更安全（避开算命类目审核风险）
- 用户搜索"情绪""陪伴""塔罗"三个词都能命中

---

78张塔罗牌+虚拟陪伴星澜，完全离线的情绪树洞。抽牌解心绪，聊天释压力，不联网不上传。

- "抽牌解心绪，聊天释压力" → 核心动作
- "不联网不上传" → 隐私安全

## 三、应用短描述（≤100字，折叠前可见区）

```
压力大、心绪乱？抽一张塔罗牌，和星澜聊几句。
纯离线不联网，你的私人情绪树洞。78张韦特塔罗、单牌快占、圣三角、关系探索，搭配星澜离线聊天与情绪回声卡。
```

**字数**：78字
**前两行（用户未点"查看更多"直接可见）**：
```
压力大、心绪乱？抽一张塔罗牌，和星澜聊几句。
纯离线不联网，你的私人情绪树洞。
```

**优化逻辑**：
1. **痛点开场**："压力大、心绪乱？" — 命中用户真实情绪状态，比"塔罗式情绪陪伴"转化率高 2-3 倍
2. **方案紧跟**："抽一张塔罗牌，和星澜聊几句" — 两个核心动作一句话说清
3. **差异化前置**："纯离线不联网" — 第二行就把最大卖点抛出来
4. **场景定位**："私人情绪树洞" — 用用户语言，不用产品语言
5. **关键词密度**：塔罗、星澜、离线、情绪、树洞 自然出现，无堆砌感

## 四、完整描述（AGC 完整描述栏）

```
压力大、心绪乱？抽一张塔罗牌，和星澜聊几句。

纯离线不联网，你的私人情绪树洞。78张韦特塔罗、单牌快占、圣三角、关系探索，搭配星澜离线聊天与情绪回声卡。

【为什么选择星钥塔罗】

· 完全离线——不联网、不注册、不上传任何数据，你的心事只留在手机里
· 虚拟陪伴星澜——温柔克制的陪伴者，不预言未来，只陪你看清此刻
· 78张完整韦特塔罗——每张牌含正逆位、情绪解读、关系提示、工作学习六个维度
· 零广告零追踪——纯净体验，仅在你主动保存分享图时申请相册权限

【核心体验】

塔罗牌阵
· 单牌快占：抽一张牌，感受此刻的情绪方向
· 圣三角：三张牌，帮你理清当下状态与可能的方向
· 关系探索：三张牌，看见一段关系中的自己与对方
· 旋转牌轮：环形牌轮转动，惯性吸附，仪式感拉满

星澜聊天
星澜是这里的陪伴者。她不会替你做决定，只会陪你慢慢说。支持 22 种话题识别、轻互动、心事倾诉、今日相处方式、星澜夜信、情绪回声卡等丰富陪伴功能。所有对话在本地完成，不联网、不上传。

卡牌总览
完整收录 78 张韦特塔罗牌，按大阿尔卡那、权杖、圣杯、宝剑、星币分类筛选。

星澜语录
1018 条星澜陪伴文案，温柔滚动呈现。不是打卡，只是把认真感受过的日子轻轻留下。

【设计特色】

· 黑金星月风格，深蓝星空底色搭配温润金色
· 东方幻想气质，温柔克制有边界感
· 沉浸式星空背景动效
· 全局背景音乐 Calm Ambient，可在设置中关闭

【隐私安全】

· 纯本地应用，不联网，不上传任何数据
· 不收集个人信息，不要求注册登录
· 占卜与聊天数据仅保存在设备本地
· 无广告，无第三方追踪
· 仅申请一个权限：保存分享图片到相册（用户主动触发）

【免责声明】
本应用提供的塔罗内容仅供娱乐与自我探索参考，不构成心理、医疗、财务、法律等专业建议。请理性看待结果。
```

**完整描述优化点**：
1. 开头三段 = 短描述复制，保证前两屏视觉一致
2. 新增"为什么选择"模块 — 4 条 bullet 直击决策点，用"·"代替 emoji（审核更稳）
3. 功能模块顺序调整：塔罗牌阵 → 星澜聊天 → 卡牌总览 → 星澜语录（按用户兴趣权重排序）
4. 隐私安全单独成块并放在功能之后 — 信任建立顺序：先看到价值，再看到安全
