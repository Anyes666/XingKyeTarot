## 二、字体层级 (Typography Scale)

| 层级 | 字号 | 字重 | 颜色 | 用途 |
|---|---|---|---|---|
| `FONT_APP_NAME` | 32px | Bold (700) | `GOLD_MAIN` | 首页品牌名 |
| `FONT_PAGE_TITLE` | 24px | Bold (700) | `TEXT_PRIMARY` | 页面标题（问心页） |
| `FONT_SECTION_TITLE` | 18px | Bold (700) | `TEXT_PRIMARY` | 结果页牌名 |
| `FONT_BODY` | 14px | Regular | `TEXT_PRIMARY` / `MOON_MUTED` | 正文、对话气泡 |
| `FONT_LABEL` | 12px | Regular | `MOON_MUTED` | 标签、提示 |
| `FONT_CAPTION` | 10px | Regular | `TEXT_DIMMED` | 免责声明 |

行高统一为 1.5~1.6 倍（如 14px → 22px 行高，12px → 18px 行高）。  
全局字间距（letter-spacing）控制在 0.5~3 之间，视场景而定。

## 六、核心组件规范

### 6.1 对话气泡 (Dialogue Bubble)

```
规格：
- 背景：backgroundBlur(15) / backdropBlur(15) + fill #1B1B30 opacity 0.68
- 圆角：16px
- 内边距：left 14 / right 14 / top 12 / bottom 12
- 最大宽度：屏幕宽度 80%
- 左对齐，不居中
- 最新条目：额外金色 1.2px 边框 + 金色圆点指示器
- 旧条目：opacity 0.78 略微降低
```

### 6.2 主 CTA 按钮 (Primary Button)

```
规格：
- 宽度：88% 屏幕宽度
- 高度：54px
- 圆角：28px
- 背景：linear-gradient(135deg, #F5D77A 0%, #E5A50A 55%, #D99A00 100%)
- 阴影：drop-shadow(0 8px 20px rgba(217,154,0,0.4))
- 文字：18px Bold，字间距 2，颜色 BG_DEEP_INDIGO
- Pressed：scale(0.95)，150ms EaseInOut
- 辅助背景文字：12px MOON_MUTED 居上 8px
```

### 6.3 情绪卡片 (Emotion Card)

```
规格：
- 网格：2 列 × 3 行
- 卡片尺寸：约 160×130
- 圆角：16px
- 默认态：fill #1B1B30 opacity 0.5 + backdropBlur(10)
- 选中态：fill 金色渐变 0.18 + stroke #E5A50A 1.5px
- 图标：16px
- 标题：15px Bold
- 副标题：9px 情绪色
- 描述：11px MOON_MUTED
- 标签：9px TEXT_DIMMED
- 右上角勾选圆点：选中时显示 r8 金色圆 + 勾
- Pressed：scale(0.97) 200ms EaseInOut
```

### 6.4 底部导航 (Bottom Tab Bar)

```
规格：
- 5 个 Tab：首页 / 历史 / [星澜按钮] / 星图 / 我的
- 高度：~90px（含 safe area）
- 背景：fill #0F0F23 opacity 0.85
- 选中态：金色 circle r14 fill #E5A50A 18%
- 图标：选中金色，未选 #94A3B8 55%
- 标签：9px，选中 Bold 金色，未选 Regular #94A3B8
- 中央星澜按钮：凸起圆形 r22，金色边框 1.5，内渐变 #F5D77A→#D99A00
- 中央按钮文字："星" 14px Bold #0F0F23
```

### 6.5 解读面板 (Interpretation Panel)

```
规格：
- 宽度：100% - 左右边距
- 圆角：12-14px
- 背景：深色渐变 #161630 → #141428
- 边框：可选金色 0.8px（金句面板）/ 仅内边线（深度解读）
- 标题：12px Bold 金色（"今日行动建议""深度解读"）
- 正文：12-14px Regular 月白
- 可展开面板默认折叠 72px，点击展开 300ms EaseOut
- 关键词标签：极光色 #5ED7D2 描边胶囊，26px 高
```

## 九、从现有 Theme 迁移对照

| 现有 Token (Theme.ets) | 升级动作 |
|---|---|
| `Theme.BG_DEEP_INDIGO` | 保持 #0F0F23，不变 |
| `Theme.GOLD_MAIN` / `Theme.GOLD_AMBER` | 明确三阶金色：LIGHT / MAIN / AMBER |
| `Theme.GOLD_LIGHT` | 明确的 #F5D77A 高光 |
| `Theme.AURORA` / `Theme.AURORA_LIGHT` | 保留 #5ED7D2，主要做关键词和标签色 |
| `Theme.CARD_BG` | 改为 #1B1B30（稍偏暖），增加 backdropBlur(15) |
| `Theme.VOID_BG` | 保留 #0A0A14，用于极端深色场景 |
| `Theme.MOON_MUTED` | 调整 #94A3B8（原值可能不一样，需对照） |
| `Theme.SPACE_*` | 调整为 8-point 系统，消除非 4 倍数的值 |
| `Theme.RADIUS_*` | 统一为 6/12/14/16/28 |

**迁移原则**：不改数据结构，只调 CSS/Theme 常量值。每调一个 Token，全量编译验证。

## 十、设计决策摘要

| 决策 | 选择 | 理由 |
|---|---|---|
| 星空背景 | 深蓝紫渐变 + 星点 + 光晕 | 保持现有气质，升级层次感 |
| 毛玻璃替代纯半透卡片 | backdropBlur(15) | 华为 HDS 推荐，提升质感 |
| 对话气泡最新条目金边 | stroke #E5A50A 1.2px | 视觉锚点，引导注意力 |
| 情绪按彩色编码 | 6 色对应 6 种情绪 | 降低认知负担，快速识别 |
| 牌面占画面约 38% | 110×176 居中 | 视觉重心不挤压解读空间 |
| CTA 按钮全宽 88% | 24px 左右边距 | 与列表/卡片对齐，触控舒适 |
| 底部导航保持 5 Tab | 中央星澜按钮凸起 | 品牌记忆点 |
