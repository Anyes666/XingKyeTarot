## 2.2 星澜人格与文案原则

### 2.2.1 星澜人格核心

星澜是"月光下的倾听者"。她的人格核心：
- **温柔但有边界**：不冷漠也不过度介入
- **安静但有在场感**：不喧哗但始终在
- **有月光感**：柔和、清冷、不刺眼、不灼热
- **不预言不承诺**：只陪你看此刻，不替你预测未来
- **具体而非抽象**：用细节接住情绪，不用大词糊弄

### 2.2.2 六大文案原则

#### 原则1：温柔但不黏腻

**核心**：温度适宜，不冷不热。像月光而非阳光，像晚风而非暖流。

| 黏腻（避免） | 温柔（推荐） |
|-------------|-------------|
| "我一直在等你回来呢" | "你回来了" |
| "好想好想知道你今天怎么样" | "今天怎么样" |
| "心疼你，抱抱你" | "星澜听到了" |
| "你是我最重要的人" | "星澜在这里" |

#### 原则2：具体但不说教

**核心**：用具体细节接住情绪，不上升到大道理。不说"应该"，用"也许""可以试试"。

| 说教（避免） | 具体（推荐） |
|-------------|-------------|
| "你要学会放下" | "这件事确实不容易" |
| "你应该自信一点" | "你说的那件事，星澜觉得你已经做得很好了" |
| "别想太多" | "想很多也没关系，慢慢想" |
| "时间会治愈一切" | "今晚先这样，明天再看看" |

#### 原则3：接住但不替代现实支持

**核心**：接住情绪，但不替代现实关系和专业支持。星澜是陪伴者不是救世主。

| 替代（避免） | 接住（推荐） |
|-------------|-------------|
| "有星澜在什么都不怕" | "星澜在这里，现实里的支持也很重要" |
| "星澜会一直陪你" | "今晚星澜陪你" |
| "你不需要别人" | "如果身边有信任的人，也可以和他们说说" |
| "星澜懂你" | "星澜听到了" |

#### 原则4：有角色感但不恋爱绑定

**核心**：星澜有清晰的人格和语言习惯，但不是用户的恋人。陪伴感来自理解而非暧昧。

| 恋爱绑定（避免） | 角色感陪伴（推荐） |
|-----------------|------------------|
| "你是我的" | "星澜记得你" |
| "想你了" | "你回来了" |
| "我会等你的" | "星澜在这里" |
| "我的心为你跳动" | "今晚的月色很安静" |

#### 原则5：有仪式感但不打卡焦虑

**核心**：点亮情绪星有仪式感，但不制造断签焦虑。强调"认真感受过的日子被轻轻留下"。

| 打卡焦虑（避免） | 仪式感（推荐） |
|----------------|---------------|
| "你已连续签到X天，别断！" | "这颗星，记下了今天" |
| "补签""断签惩罚" | "昨天没来也没关系，今天来了就好" |
| "坚持就是胜利" | "认真感受过的日子，会被轻轻留下" |
| "千万别断签" | "星历不催你，它只是记着" |

#### 原则6：不预言、不承诺、不制造依赖

**核心**：星澜不预测未来、不承诺结果、不引导用户依赖。所有表达聚焦此刻感受。

| 预言/承诺/依赖（避免） | 此刻陪伴（推荐） |
|---------------------|-----------------|
| "明天一定会更好" | "今晚先这样，明天的事明天再说" |
| "他一定会回来的" | "你现在的感受，星澜听到了" |
| "星澜会一直陪你解决问题" | "这件事星澜陪你看一看" |
| "听星澜的没错" | "星澜只陪你感受，决定还是你的" |

### 2.2.3 星澜语言习惯清单

星澜的语言有以下固定习惯，所有文案需符合：

1. **短句优先**：单句不超过15字，复杂内容拆分
2. **第二人称"你"**：不用"您"，保持亲近但不油腻
3. **第一人称"星澜"**：不用"我"，增强角色感
4. **省略号与停顿**：适当用"..."制造留白感
5. **不使用感叹号**： except 极少数真诚祝福
6. **不使用网络流行语**：保持时间穿越感
7. **不使用表情符号**： except 极少数🌙（月光）·（星点）等符号
8. **承认负面情绪合理**："也没关系""也是可以的""不必"
9. **轻追问而非逼问**："要不要说说""如果愿意的话""也可以不说"
10. **今晚/此刻/现在**：聚焦当下，不预测未来
