# 星钥塔罗 截图拍摄指南

## 截图要求

| 项目 | 要求 |
|------|------|
| 数量 | 至少 3 张，建议 5 张 |
| 方向 | 竖屏 |
| 分辨率 | ≥ 1080×1920（建议 1080×2340 适配全面屏） |
| 格式 | PNG 或 JPG |
| 单张大小 | ≤ 5MB |
| 内容 | 必须为真实应用界面截图，不得使用设计稿 |

---

## 建议截图清单（5张）

### 截图 1：首页 — 今日一占

**页面路径**：首页（Index）

**展示内容**：
- 应用标题"星钥塔罗"
- Slogan"探索内心，点亮星光"
- 今日一占卡片（牌名 + 关键词）
- "开始占卜"金色按钮
- 星空背景

**拍摄要点**：
- 确保今日运势已加载完成（非 Loading 状态）
- 展示完整的金色按钮和底部免责提示

## 截图拍摄操作步骤

1. 在 DevEco Studio 中使用模拟器或真机运行应用
2. 完成新手引导和免责声明确认
3. 依次导航到上述页面
4. 使用系统截屏功能截图（真机：电源键 + 音量下键）
5. 检查截图清晰度和内容完整性
6. 如需裁剪，保持竖屏比例

## 备选方案

如无法真机截图，可使用以下方式：
- DevEco Studio 模拟器截屏
- 在模拟器中运行后使用系统截屏功能

## 注意事项

- 截图中不得包含个人隐私信息
- 截图必须为应用真实运行界面，不可使用 mockup
- 如截图中包含加载状态（Loading），需等待加载完成后再截
- 建议在深色环境下截图以展现星空效果（应用本身为深色主题）
