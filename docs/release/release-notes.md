# 星钥塔罗 版本说明

## v1.0.0 （2026-06-22）— 首次发布

### 新功能

- **圣三角占卜**：经典圣三角牌阵，三张牌对应"过去 · 现在 · 未来"，逐张翻牌交互体验
- **每日一占**：每天抽取一张塔罗牌，提供关键词与运势总结，凌晨6点自动刷新
- **桌面卡片**：2x2 尺寸桌面卡片，无需打开应用即可查看每日运势
- **占卜解读**：每张牌提供正逆位关键词与详细解读文本
- **分享功能**：一键复制占卜结果，可粘贴至微信/朋友圈等平台分享
- **新手引导**：3 页滑动引导，帮助新用户了解应用功能
- **免责声明**：首次使用展示免责声明弹窗，结果页固定展示"仅供娱乐"提示

### 设计

- 新中式星空风格：深蓝紫底色 + 温润金色点缀 + 青绿极光辅助
- 沉浸式星空背景动效
- 精致的塔罗牌翻牌动画

### 技术规格

- 兼容系统：HarmonyOS 5.0.0(12) 及以上
- 目标系统：HarmonyOS 6.1.1(24)
- 支持设备：手机
- 应用包名：com.xingkey.tarot
- 纯本地应用，无需网络，无广告，无第三方SDK
