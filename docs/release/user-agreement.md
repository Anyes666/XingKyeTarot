## 一、服务说明

本应用是一款塔罗占卜与自我探索工具，提供以下功能：

1. **圣三角占卜**：基于塔罗牌圣三角牌阵的占卜与解读
2. **每日一占**：每日抽取一张塔罗牌并提供运势参考
3. **桌面卡片**：在设备桌面展示每日运势卡片
4. **分享功能**：将占卜结果以文本形式复制分享

本应用为纯本地离线应用，无需网络连接即可使用全部功能。
