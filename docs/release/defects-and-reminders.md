## ⚠️ 版本号更新提醒

**每次发版前必须更新版本号。**

修改位置：`entry/src/main/ets/common/LegalDocuments.ets`

```typescript
static readonly APP_VERSION: string = 'v1.1.0';  // ← 改这里
```

该版本号会注入「用户协议」和「隐私政策」的标题下方，App Store 审核会检查版本号一致性。
