一、服务说明

本应用是一款塔罗占卜与自我探索工具，提供以下功能：

  • 圣三角占卜：基于塔罗牌圣三角牌阵的占卜与解读
  • 每日一占：每日抽取一张塔罗牌并提供运势参考
  • 桌面卡片：在设备桌面展示每日运势卡片
  • 分享功能：将占卜结果生成图片保存到相册，并在保存失败时提供剪贴板文字兜底

本应用为纯本地离线应用，无需网络连接即可使用全部功能。
