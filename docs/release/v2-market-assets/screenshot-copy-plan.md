## 截图加工说明

1. 叠加文案使用半透明黑色底条 + 白色文字
2. 标题字号大于副文案
3. 文案位置：底部 1/3 区域，不遮挡核心内容
4. 字体：无衬线体（如思源黑体），与 App 风格一致
5. 颜色：白色文字 + 金色标题点缀（可选）

## 注意事项

- 截图中不得包含个人隐私信息
- 必须为应用真实运行界面
- 5 张截图应展示不同功能页面，避免重复
- 深色主题截图效果更佳
