# 星钥塔罗 — 应用描述

## 应用名称

星钥塔罗

## 简短描述（≤80字）

新中式星空风格塔罗占卜工具。圣三角牌阵解读过去现在未来，每日一占桌面卡片，探索内心，点亮星光。结果仅供娱乐。

## 完整描述

星钥塔罗是一款以新中式星空美学为基调的塔罗占卜与自我探索应用。我们相信每一张牌都是一面镜子，帮助你看见内心的不同面向。

【核心功能】

**圣三角占卜**
采用经典的圣三角牌阵，三张牌分别对应"过去 · 现在 · 未来"三个时间维度。逐张翻牌的仪式感体验，帮你理清当下的状态与可能的方向。每张牌提供正逆位关键词与详细解读，支持复制分享占卜结果。

**每日一占**
每天为你抽取一张塔罗牌，配以关键词和运势总结，适合作为一天的灵感提醒。每日凌晨6点自动刷新，也可随时查看当日运势详情。

**桌面卡片**
支持添加2x2桌面卡片，无需打开应用即可在桌面上查看每日一占的牌面与关键词，每日定时自动更新。

【设计特色】

- 新中式星空风格：深蓝紫底色搭配温润金色点缀，青绿极光辅助配色
- 沉浸式星空背景动效
- 精致的翻牌交互体验
- 完整的新手引导流程

【隐私安全】

- 纯本地应用，不联网，不上传任何数据
- 不收集任何个人信息，不要求注册登录
- 占卜数据仅保存在你的设备本地
- 无广告，无第三方追踪SDK

本应用提供的塔罗内容仅供娱乐和自我探索参考，不构成心理、医疗、财务、法律等专业建议。请理性看待结果。
