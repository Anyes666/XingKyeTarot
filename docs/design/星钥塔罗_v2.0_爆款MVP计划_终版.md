# 星钥塔罗 v2.0 爆款 MVP 计划 · 终版

> **产品定义**: 塔罗式情绪陪伴 App，不是占卜工具。
> **一句话**: 抽一张牌，看见你没说出口的心事。
> **评分**: 9.0/10 → 10.0 需要：更快第一屏、更狠分享卡、更早给反馈。

---

## 零、最小爆款闭环（8 张牌版）

```
首页一层问心 → 单牌快占 → 金句击中 → 月见追问（第二层）→ 分享卡 + 情绪星点 → 明天回来
```

**首次体验不超过 3 步**: 选情绪 → 抽牌 → 被击中。

## 二、第一版只做 8 张牌

### 为什么 8 张

22 张大阿卡纳一起写 = 内容生产黑洞。先精选 8 张高频牌精修，验证"击中感"。

| 牌 | 覆盖的心理场景 |
|----|--------------|
| 愚人 | 新开始、跳槽、新关系 |
| 女祭司 | 暧昧不明、信息不透明、直觉提醒 |
| 恋人 | 关系选择、价值观匹配 |
| 战车 | 行动力、突破、掌控 |
| 隐士 | 独处思考、需要沉淀 |
| 命运之轮 | 变化、转机、周期起落 |
| 死神 | 放下、结束、蜕变 |
| 月亮 | 内耗、猜测、安全感不足 |

这 8 张覆盖了用户最常见的 8 种心理状态，先把它们做到"一击必中"，再扩展到 22 张。

## 三、P0-1：首页一层问心

### 设计

```
"You打开App → 一句问候 → 6个选项 → 选完直接抽牌"
```

**问心文案**（温暖但不煽情）:

> 你现在，最想被哪件事接住？

**6 个选项**（场景化情绪，不是分类名词）:

```
💔 他/她忽冷忽热，我很内耗
💕 我想知道这段关系值不值得
😮‍💨 我最近工作很累，想逃离
🔀 我正在纠结一个选择
🫥 我感觉自己能量很低
🫂 我只是想被安慰一下
```

**为什么首次只一层**: 新用户耐心极短，打开 App 只想要"快点说中我"。第二层追问移到结果页后，作为月见追问的一部分。

### 选完后直接抽牌

- 默认单牌快占（最快击中）
- 复杂牌阵入口放在结果页底部「换一个牌阵试试」
- 把 `SELECT_TOPIC` 阶段砍掉——从问心直接进抽牌

## 四、P0-2：结果页第一屏金句卡

### 一屏之内必须包含

```
┌─────────────────────────────────┐
│                                 │
│          🃏 月亮正位             │
│                                 │
│  ┌─────────────────────────┐   │
│  │                         │   │
│  │   "你不是想太多，          │   │
│  │    你只是太久没有安全感。"   │   │
│  │                         │   │
│  └─────────────────────────┘   │
│                                 │
│   📌 情绪关键词                  │
│   信息不透明 · 反复猜测 · 安全感不足 │
│                                 │
│   ✅ 今日行动                    │
│   今天先不要逼自己做决定            │
│                                 │
│  ┌──────────┐ ┌──────────┐     │
│  │ 💬 追问月见│ │ 📤 分享卡片│     │
│  └──────────┘ └──────────┘     │
│                                 │
│  ⬇ 下滑看深度解读                 │
└─────────────────────────────────┘
```

### 黄金文案规则

> **点破，但不审判。扎心，但不制造焦虑。共情，但不替用户下结论。**

| ❌ 不要写 | ✅ 写成 |
|----------|--------|
| 他不会回来 | 这段关系里，你可能正在反复等待一个确定的回应 |
| 这段关系没结果 | 与其继续猜测，不如先确认自己的底线和需求 |
| 你们不合适 | 你们之间可能缺少一次真正清晰的沟通 |
| 他不爱你 | 你值得一个不让你反复猜的人 |
| 你太敏感了 | 你的直觉正在提醒你关注一些被忽略的细节 |

涉及感情的文案，不做终局审判，只做现状描述 + 行动引导。

## 五、P0-3：月见追问（第二层个性化入口）

### 为什么是 P0

用户看完结果最自然的动作 = 继续问。"月见追问"既是留存引擎，也是最大付费点。

### 交互

结果页金句屏 → 点击「追问月见」→ 弹出：

```
┌─────────────────────────────────┐
│                                 │
│  月见想问你：                     │
│                                 │
│  ┌─────────────────────────┐   │
│  │ "你更难受的是'不确定'，     │   │
│  │  还是'明明知道却舍不得'？"   │   │
│  └─────────────────────────┘   │
│                                 │
│  😔 不确定，心里没底              │
│  💔 明明知道，但舍不得             │
│  🤷 两个都有                     │
│                                 │
└─────────────────────────────────┘
```

选择后 → 月见定向回应（不抽新牌，基于已抽牌 + 问心选择 + 追问选择拼接）

### 付费节奏

- 前 3 次追问免费（建立信任）
- 之后：每天免费 1 次，会员（¥6/月）无限追问 + 追问历史
- 保护期结束后，第一次追问超出时弹窗: "月见可以陪你一直聊下去"


**传播场景**: 个人简介、自我表达、社交货币

## 七、P0-5：情绪星点（Day 1 即时反馈）

### 为什么不等 7 天

用户第一天看不到完整结果，"1/7"没有爽感。需要在第一天就给一个小奖励。

### 阶梯反馈节奏

```
Day 1: 🌟 你的第一颗情绪星已点亮：关系边界
       再记录 6 天，生成你的专属情绪星图。
       [视觉：一颗金色星星在星图上亮起]

Day 3: 🌟🌟 已点亮 3 颗星
       最近反复出现的关键词：关系边界 · 期待回应
       月见想说：你好像一直在等一个清晰的说法。

Day 5: 🌟🌟🌟🌟🌟 已点亮 5 颗星
       发现重复情绪模式：内耗型等待
       每次抽牌都在问"ta在想什么"，但真正的问题可能是"你在怕什么"。

Day 7: ✨ 你的情绪星图已解锁！
       完整星图 + 情绪人格 + 月见长回应
       [分享我的星图]
```

### 数据结构

```typescript
export interface StarPoint {
  date: string;
  cardId: string;
  emotionTag: string;     // '关系边界' | '自我怀疑' | '期待回应' ...
  starPosition: { x: number; y: number };  // 在星图 Canvas 上的坐标
}

export interface EmotionStarMap {
  totalDays: number;
  starPoints: StarPoint[];
  topThemes: Array<{ theme: string; count: number }>;
  personalityType: string;  // '温柔内耗型' | '清醒忍耐型' | '高敏感知型' | '慢热重启型'
  moonInsight: string;
}
```

### 情绪人格类型（4 种，有传播感）

| 类型 | 特征 | 月见一句话 |
|------|------|-----------|
| 温柔内耗型 | 向内消耗，反复想同一件事 | "你不是太敏感，你只是太久没有被认真回应。" |
| 清醒忍耐型 | 知道问题但选择扛着 | "你的理性在保护你，但偶尔也让情绪透透气。" |
| 高敏感知型 | 能察觉细微的情绪变化 | "你的细腻不是负担，是你理解世界的方式。" |
| 慢热重启型 | 需要时间恢复能量 | "慢慢来，不是所有答案都要今天找到。" |

## 十、隐私设计原则

> 星钥塔罗要给用户的安全感不只是文案上的，也包括数据上的。

### 隐私风险矩阵

| 模块 | 隐私类型 | 风险等级 | 策略 |
|------|---------|---------|------|
| 一层问心 | 情绪偏好 | 中 | 本地存储，不绑账号 |
| 抽牌结果 | 行为记录 | 低-中 | 本地优先 |
| 月见追问 | 私密表达 | **高** | 透明告知 + 仅用于本次生成 + 可删除 |
| 情绪星图 | 用户画像 | 中-高 | 本地优先，云同步单独授权 |
| 分享卡 | 主动分享 | 低 | 不带身份信息、不埋追踪 |
| 会员支付 | 交易数据 | 中 | 最小必要订单信息 |
| 推送提醒 | 行为触发 | 中 | 单独授权、可关闭 |

### MVP 六条硬规则

**1. 游客可完整体验一次**
- 打开 App → 问心 → 抽牌 → 看结果 → 追问一次 → 生成分享卡 → 点亮情绪星
- 不强制登录、不收手机号、不收定位、不读通讯录、不读相册
- 等用户想保存历史 / 同步星图 / 开会员时，再引导登录

**2. 情绪记录默认本地保存**
- 抽牌历史、情绪星点、问心选择 → `Preferences` 本地存储
- 给用户一句安心提示：*「你的问心记录默认只保存在本机，可随时删除。」*
- 开启云同步时单独弹窗确认：*「开启后，你的抽牌记录将用于跨设备同步。」*

**3. 月见追问透明告知**
- 追问输入框 / 选择区上方写明：*「你选择的内容仅用于生成本次解读，不会公开展示。」*
- 追问内容不进训练数据、不做广告画像
- 所有追问记录可一键清空

**4. 分享卡不嵌入身份信息**
- 卡片只含：牌名 + 金句 + 关键词 + 日期 + 「星钥塔罗 · 月见」水印
- 不带用户 ID、昵称、头像、定位
- 不嵌入隐藏追踪参数
- 每一步由用户主动触发：生成分享卡 → 保存到相册 → 复制文案

**5. 一键清除记录**
- 设置页提供「隐私与记录管理」入口：
  - 清空抽牌历史
  - 清空追问记录
  - 清空情绪星图
  - 导出记录
  - 删除账号（含云端数据）

**6. 不做广告画像**
- 情绪标签**不**用于第三方广告投放
- 情绪标签**不**用于情感课程 / 婚恋服务 / 心理咨询等精准推荐
- 可用于产品内体验优化（如「你最近常出现的关键词是关系边界」）
- 隐私政策用人话写，不用法务模板

### MVP 权限清单

| 权限 | 是否申请 | 原因 |
|------|---------|------|
| 读取相册 | ❌ 不申请 | 只写不读 |
| 写入相册 | ✅ 保存分享卡时申请 | 用户主动触发后 |
| 定位 | ❌ 不申请 | 完全不需要 |
| 通讯录 | ❌ 不申请 | 完全不需要 |
| 麦克风 | ❌ 不申请 | MVP 不做语音 |
| 摄像头 | ❌ 不申请 | MVP 不做拍照 |
| 剪贴板 | ❌ 不自动读取 | 仅复制文案时被动写入 |
| 设备标识 / 广告 ID | ❌ 不用 | MVP 不需要 |
| 联网 | ✅ | 抽牌和追问需服务端 |

## 十一、三个验证指标（决定是否加码）

在扩展 22 张牌之前，先用 8 张牌版跑通 3 个问题：

| # | 问题 | 验证方式 | 及格线 |
|---|------|----------|--------|
| 1 | 用户在 30 秒内说出"有点准"？ | 首次抽牌到金句屏的停留 | > 15 秒 |
| 2 | 用户愿意点"追问月见"？ | 追问按钮点击率 | > 30% |
| 3 | 用户愿意保存或分享情绪卡？ | 分享/保存率 | > 8% |

三项跑通 → 加码做 22 张牌 + 关系温度牌阵 + 7 天星图。
