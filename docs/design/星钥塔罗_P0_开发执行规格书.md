# 星钥塔罗 P0 开发执行规格书

> **性质**: 开发执行规格书（不是产品方案）
> **目标**: Agent 可按本文档逐屏实现 MVP，无需再追问产品逻辑
> **对应原型**: Ardot 画布 `696852272011003`（7 屏高保真原型）
> **对应方案**: `星钥塔罗_v2.0_爆款MVP计划_终版.md`
> **技术栈**: HarmonyOS ArkTS + Stage 模型 + Preferences 本地存储
> **执行前提**: Agent 在动第一行代码前，必须先读本补丁（覆盖正文中可能存在的冲突表述）

## 🔴 Agent 执行补丁（优先级最高）

> 以下 10 条是 P0 开发的硬约束。正文中如有任何表述与补丁冲突，以补丁为准。

| # | 补丁内容 | 说明 |
|---|---------|------|
| 1 | **P0 默认跳过 Onboarding** | EntryAbility 启动后直接进入 `pages/Index`（AskHeartPage）。OnboardingPage 保留但不接入主流程。正式上架版再加。 |
| 2 | **P0 不做登录、会员、支付、后端 AI、云同步** | 纯本地应用。所有数据打包在 `data/` 目录。 |
| 3 | **月见追问采用本地结构化文案匹配** | 不调用大模型。追问回应从 `FollowUpQuestions.ets` 预写文案库中匹配。 |
| 4 | **游客每天可追问 1 次** | 用 `Preferences` 本地记录每日追问计数。"前 3 次免费"是后续商业化策略文案，不在 P0 实现。 |
| 5 | **cardWeights 只引用 MVP8 张牌** | 只允许 `major_00_fool / major_02_high_priestess / major_06_lovers / major_07_chariot / major_09_hermit / major_10_wheel_of_fortune / major_13_death / major_18_moon`。抽到非 MVP8 cardId 时 fallback 到 `major_18_moon`。 |
| 6 | **所有前台文案去宗教化、去神秘学绝对化** | 禁止出现：亚当、夏娃、伊甸园、天使、拉斐尔、希伯来字母 Y-H-V-H、斯芬克斯、老天安排、宇宙帮你、改命、转运、宿命、神谕、必然发生。用"心理隐喻"表达替代。 |
| 7 | **P0 默认不申请 INTERNET** | 不接后端，不联网。隐私感更强，适合快速原型测试。接入后端时再加此权限。 |
| 8 | **所有文案从 data 层读取** | 不允许写在页面组件中。页面只做渲染和交互。 |
| 9 | **每完成一个页面必须截图对比 Ardot 原型** | 截图后与原型视觉比对，再进入下一页。 |
| 10 | **不允许自行添加 P1/P2/P3 功能** | 会员、社交分享、AI 追问、星图分析、消息推送等一律不做。 |

## 二、AppStage 页面状态机

```
┌──────────────────────────────────────────────────────────────────────┐
│                           AppStage 状态机                              │
├──────────────┬───────────────────────────┬───────────────────────────┤
│    阶段       │        进入条件            │        出口               │
├──────────────┼───────────────────────────┼───────────────────────────┤
│ ONBOARDING   │ 保留但不接入主流程（P0 跳过）    │ -                         │
│ ASK_HEART    │ 启动即进入 / 再次打开             │ 选情绪 → DRAWING_CARD      │
│ DRAWING_CARD │ 问心选择提交后               │ 动画结束 → RESULT          │
│ RESULT       │ 抽牌动画完成                 │ 点追问 → MOON_FOLLOW_UP   │
│              │                            │ 点分享 → SHARE_CARD        │
│              │                            │ 点星点 → STAR_POINT        │
│              │                            │ 返回 → ASK_HEART           │
│ MOON_FOLLOW_UP│ 结果页点"追问月见"           │ 选完 → 月见回应弹出         │
│              │                            │ 关闭 → RESULT              │
│ SHARE_CARD   │ 结果页点"分享"               │ 保存/分享 → RESULT         │
│ STAR_POINT   │ 结果页点"情绪星点"            │ 查看完 → RESULT            │
│ PRIVACY      │ 设置 → 隐私与记录管理         │ 返回 → ASK_HEART           │
│ HOME         │ 底部导航首页（v2.1）          │ -                         │
└──────────────┴───────────────────────────┴───────────────────────────┘
```

### 路由映射

| AppStage | Route | EntryAbility 启动判断 |
|----------|-------|----------------------|
| ONBOARDING | `pages/OnboardingPage` | P0 跳过，保留文件不接入路由 |
| ASK_HEART | `pages/Index` | 默认首页 |
| DRAWING_CARD | `pages/CardDrawPage` | router.pushUrl |
| RESULT | `pages/ResultGoldenPage` | router.replaceUrl（不可回退到动画） |
| MOON_FOLLOW_UP | `pages/MoonFollowUpPage` | router.pushUrl（可回退到结果） |
| SHARE_CARD | `pages/ShareCardPage` | router.pushUrl |
| STAR_POINT | `pages/StarPointPage` | router.pushUrl |
| PRIVACY | `pages/PrivacySettingsPage` | router.pushUrl |

## 五、8 张牌完整结构化内容

> 每张牌对应 6 种情绪场景（heartEmotionTag → ScenarioContent），每张牌 3 组追问（FollowUpQuestion × 3）。

### 5.1 愚人（The Fool · major_00_fool）

**关联情绪人格**: 慢热重启型

| emotionTag | goldenSentence | emotionKeywords | todayAction |
|------------|---------------|-----------------|-------------|
| 关系内耗 | "你反复琢磨的那些细节，可能只是因为你太想抓住什么。" | 过度投入 · 害怕错过 · 反复确认 | 今天先别逼自己判断"对错"，让事情自然流动 |
| 关系评估 | "有些路不用知道终点再出发，先走一步，答案在路上。" | 勇敢尝试 · 不确定感 · 新开始 | 给关系一个"试试看"的机会，不预设结果 |
| 职场疲惫 | "你不是想逃避，你只是太久没有为自己做选择了。" | 倦怠感 · 渴望自由 · 重新选择 | 写下你真正想做的三件事，挑一件本周开始 |
| 选择纠结 | "想不清楚的时候，不如先跳一步。答案不在脑子里，在行动里。" | 犹豫 · 勇气 · 突破 | 抛硬币——不是为了看结果，是为了在硬币飞起时，发现自己在期待哪一面 |
| 能量低落 | "一个全新的开始，不需要你准备好——只需要你愿意。" | 空白 · 可能 · 信任 | 今天只做一件小事：走出房间，呼吸新鲜空气 |
| 寻求安慰 | "你啊，心里那个小孩想出去玩。不是要你勇敢，是要你轻装上阵。" | 轻盈 · 直觉 · 童心 | 对自己说一句：今天可以什么都不做好 |

**追问 × 3**:

1. **"你在怕的是'走错'，还是怕'没人陪你走'？"**
   - 😰 怕走错 → "走错的路也是路。愚人从来不回头看悬崖，他只看见前方的天空。"
   - 🫂 怕没人陪 → "你不是一个人。愚人身边那条小狗，就是生活给你的提醒——总有人默默跟着你。"
   - 🤷 都有 → "怕就对了。怕说明你在乎。但不要因为怕，就停在原地。愚人迈出那一步的时候，也没人给他保证。"

2. **"如果没有任何限制，你内心最想做的那件事是什么？"**
   - 💡 有明确想法 → "那就去做。不是明天，不是下周——今天就做第一小步。愚人牌在告诉你：时机到了。"
   - 🤔 暂时没想到 → "没关系。不知道去哪儿也是一种自由。允许自己暂时没有方向，反而会有新的东西进来。"
   - 😞 想到了但觉得不现实 → "所有的'不现实'都是经验在说话。但经验是过去的事。愚人不在乎过去——他只在乎脚下这一步。"

3. **"你通常怎么面对'未知'？"**

## 附录 B：cardWeights 加权规则

6 个情绪标签分别对应 8 张牌的权重分布。权重越高的牌在该情绪下越容易被抽到。Agent 实现时的抽牌逻辑：

```typescript
// 伪代码
function drawCard(emotionTag: string): string {
  const weights = HEART_OPTIONS[emotionTag].cardWeights;
  // weights 例如: { major_18_moon: 30, major_02_high_priestess: 15, major_06_lovers: 10, ... }
  // ⚠️ 只允许引用 MVP8 张牌（见执行补丁第 5 条）。非 MVP8 cardId → fallback major_18_moon
  // 加权随机选择一张牌
  return weightedRandom(weights);
}
```
