## 当前方向

更偏向中国女性用户审美的方向是：黑金星月边框、精致幻想人物、深蓝星空、月相、钥匙、白马/星光动物等符号。

正式牌面建议不在图片内生成文字，牌名、编号、正逆位由 App 统一叠加。

## 后续生成规则

- 风格主线：黑金星月、深蓝星空、精致幻想人物、中国女性用户友好。
- 必备元素：月相、星轨、钥匙、金色边框、塔罗核心符号。
- 禁止项：图片内不生成文字、不做恐怖血腥、不做过度暴露、不做太游戏化/机甲化。
- 命名规则：`v编号-主题-风格说明.png`。
- 每次生成后必须同步本 README。
